# Registro de Empleados

## Descripción
Este proyecto consiste en desarrollar una aplicación de escritorio que permite registrar la información de un empleado, verificando que todos los campos obligatorios estén completos antes de guardar los datos.

La aplicación utiliza cuadros de diálogo para que el usuario pueda seleccionar el nombre y la ubicación del archivo de texto donde se almacenará la información.

El sistema muestra mensajes de confirmación al momento de generar el archivo y también cuando el usuario intenta cerrar la aplicación. Además, ofrece la opción de abrir el archivo guardado para visualizar los datos registrados.

La información de los empleados se guarda en formato de tabla dentro del archivo de texto, y la interfaz del programa cuenta con un diseño sencillo que aplica criterios básicos de usabilidad y accesibilidad.

---

## Tecnologías Utilizadas
- Lenguaje: C#
- Framework: .NET (Windows Forms)
- Entorno de desarrollo: Visual Studio 2022
- Almacenamiento: Archivo de texto (.txt)

---

## Funcionalidades
- Registro de datos de empleados.
- Validación de campos obligatorios.
- Selección de ubicación para guardar el archivo.
- Generación de archivo de texto con los datos.
- Mensajes de confirmación y alerta.
- Apertura del archivo guardado.

---

## Uso o Ejecución
1. Clonar el repositorio desde GitHub.
2. Abrir el archivo `.sln` en **Visual Studio 2022**.
3. Compilar el proyecto.
4. Ejecutar la aplicación presionando **F5** o el botón **Iniciar**.
5. Ingresar los datos del empleado y guardar la información.
