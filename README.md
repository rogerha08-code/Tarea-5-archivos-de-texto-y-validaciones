El proyecto consiste en desarrollar una aplicación que permita capturar los datos de un empleado,
validando que todos los campos obligatorios estén completos antes de guardar la información.
La aplicación debe utilizar cuadros de diálogo para que el usuario pueda elegir
el nombre y la ubicación del archivo de texto donde se almacenarán los datos.
El sistema mostrará mensajes de alerta al generar el archivo y al intentar salir de la aplicación, además de permitir abrir el archivo guardado.
a información de los empleados se guardará en formato de tabla,
y la interfaz contará con un diseño sencillo que aplique criterios básicos de usabilidad y accesibilidad.
